# คู่มือสมัครและสร้างเครื่องเซิร์ฟเวอร์ฟรีบน Oracle Cloud (Always Free)

Oracle Cloud ให้สิทธิใช้งานเซิร์ฟเวอร์ฟรีตลอดชีพ (Always Free) ซึ่งคุ้มค่ามากๆ และเราสามารถเลือกโซนในเอเชีย (เช่น สิงคโปร์ หรือ ญี่ปุ่น) เพื่อหลีกเลี่ยงปัญหาการโดนแบนจาก Binance ได้ครับ

---

## ขั้นตอนที่ 1: การสมัครสมาชิก (จุดที่ยากที่สุด)
> **⚠️ คำเตือนสำคัญ:** ระบบกันสแปมของ Oracle ค่อนข้างเข้มงวดมาก หลายคนมักจะสมัครไม่ผ่าน (โดนปฏิเสธบัตรเครดิต) แนะนำให้ใช้บัตรเครดิตจริง (บัตรเดบิตหรือ TrueMoney มักจะไม่ผ่าน) และกรอกข้อมูลให้ตรงกับที่อยู่จัดส่งบิลของบัตรเครดิตเป๊ะๆ ครับ

1. เข้าไปที่เว็บ [Oracle Cloud Free Tier](https://www.oracle.com/cloud/free/) แล้วคลิก **"Start for free"**
2. กรอกข้อมูลส่วนตัว อีเมล รหัสผ่าน ให้ครบถ้วน
3. **การเลือก Home Region:** สำคัญมาก! ให้เลือก **Singapore** หรือ **Japan (Tokyo/Osaka)** เพราะเป็นโซนเอเชียที่เชื่อมต่อกับ Binance ได้รวดเร็ว และเน็ตเวิร์คไม่บล็อค (ถ้าเลือกผิดจะเปลี่ยนทีหลังไม่ได้)
4. ยืนยันตัวตนด้วยบัตรเครดิต (ระบบจะตัดเงินประมาณ 30 กว่าบาทเพื่อเช็คว่าบัตรมีเงินจริง แล้วจะคืนให้ทีหลัง)
5. ถ้าระบบขึ้นว่าสำเร็จ ให้รอรับอีเมลยืนยันการสร้างบัญชี (อาจใช้เวลา 5-15 นาที)

## ขั้นตอนที่ 2: การสร้างเครื่อง VM (เซิร์ฟเวอร์)
เมื่อเข้าสู่ระบบ (Dashboard) ของ Oracle Cloud ได้แล้ว:
1. กดที่เมนูสามขีดซ้ายบน -> **Compute** -> **Instances**
2. กดปุ่ม **Create Instance**
3. ตั้งชื่อเครื่องอะไรก็ได้ในช่อง Name
4. **Image and Shape (สำคัญมาก!):**
   - กดปุ่ม `Edit` ตรง Image and Shape
   - **Image:** เปลี่ยนเป็น `Ubuntu` (แนะนำเวอร์ชัน 22.04)
   - **Shape:** เปลี่ยนเป็น `Ampere` (ARM) แล้วเลือก `VM.Standard.A1.Flex`
   - **ติ๊กเลือก OCPU:** เลื่อนแถบไปที่ `1` หรือ `2` (สูงสุดได้ 4) และ **RAM** เป็น `6GB` หรือ `12GB` (สูงสุด 24GB) 
   *(หมายเหตุ: แถบสีเขียวจะเขียนว่า "Always Free Eligible" แปลว่าฟรีแน่นอน)*
5. **Networking:** ปล่อยค่าดั้งเดิมไว้ (แต่ให้จำไว้ว่ามี Public IP)
6. **Add SSH keys (สำคัญสุดๆ สำคัญกว่าชีวิต!):**
   - ให้เลือก **"Save private key"** และ **"Save public key"** ดาวน์โหลดเก็บไว้ในคอมพิวเตอร์ของคุณ **ห้ามทำหายเด็ดขาด** เพราะจะใช้ไฟล์นี้แทนรหัสผ่านในการเข้าเครื่อง
7. **Boot volume:** ปรับขนาดได้สูงสุด `200 GB` (ฟรี) แต่ตั้งไว้สัก `50 GB` ก็เหลือเฟือแล้วครับ
8. กดปุ่ม **Create** ด้านล่างสุด แล้วรอจนกว่าเครื่องจะขึ้นสถานะแถบสีเขียว (Running)

## ขั้นตอนที่ 3: เปิด Port 8000 (เหมือนฝั่ง Google)
บน Oracle ต้องเปิด Port 2 ที่ครับ (บนเว็บ และ ในเครื่อง)

**ก. เปิดบนเว็บ Oracle:**
1. ในหน้าของ Instance ที่เพิ่งสร้าง ให้เลื่อนลงมากดคลิกที่ชื่อ **Subnet**
2. คลิกที่ **Security Lists** (มักจะชื่อ Default Security List...)
3. กด **Add Ingress Rules**
   - Source CIDR: `0.0.0.0/0`
   - Destination Port Range: `8000`
   - กด **Add Ingress Rules** เซฟให้เรียบร้อย

## ขั้นตอนที่ 4: การรีโมทเข้าเครื่อง (SSH)
เนื่องจาก Oracle ไม่ได้มีปุ่มให้กดเปิดหน้าจอดำบนเว็บแบบ Google เราต้องใช้โปรแกรมในคอมพิวเตอร์ของเราเข้าเองครับ:
1. เปิดโปรแกรม `Windows PowerShell` ในคอมของคุณ
2. พิมพ์คำสั่งเพื่อเชื่อมต่อไปยังเครื่อง (ใช้ไฟล์ Private Key ที่โหลดมาจากข้อ 2.6):
   ```bash
   ssh -i "C:\ที่เก็บไฟล์\ssh-key-2026-xx-xx.key" ubuntu@เลข_IP_ของเครื่อง_Oracle
   ```
   *(ตัวอย่าง: `ssh -i "C:\Users\son11\Downloads\ssh-key.key" ubuntu@123.45.67.89`)*
3. ถ้าระบบถาม (yes/no) ให้พิมพ์ `yes` แล้ว Enter

---

## ขั้นตอนที่ 5: เอาโค้ดลงเครื่อง
พอเข้ามาในหน้าจอดำของ Oracle ได้แล้ว ก็ทำเหมือนสเต็ปของ Google Cloud ได้เลยครับ!
1. อัปเดตและลงโปรแกรม (เนื่องจากเครื่องเป็น ARM บางที Python จะติดตั้งไว้แล้ว):
   ```bash
   sudo apt update && sudo apt upgrade -y
   sudo apt install python3 python3-pip python3-venv git screen -y
   ```
2. เปิด Port 8000 ภายในเครื่อง (เฉพาะบน Oracle ต้องทำบรรทัดนี้ด้วย):
   ```bash
   sudo iptables -I INPUT 6 -m state --state NEW -p tcp --dport 8000 -j ACCEPT
   sudo netfilter-persistent save
   ```
3. โหลดโค้ดและรันบอท:
   ```bash
   git clone https://github.com/son11930/binance-trade-bot binancetrade
   cd binancetrade
   pip3 install -r requirements.txt
   nano .env  # (ใส่คีย์ API)
   chmod +x start.sh
   screen -S bot
   ./start.sh
   ```

พอบอทรันเสร็จ ก็สามารถเปิดมือถือเข้าเว็บ `http://เลข_IP_Oracle:8000` ได้เลยครับ จบปิ๊ง! 🚀
