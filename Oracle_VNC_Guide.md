# คู่มือสมัคร สร้างเซิร์ฟเวอร์ฟรี และตั้งค่าหน้าจอรีโมท (VNC/RDP) บน Oracle Cloud (Always Free)

Oracle Cloud ให้สิทธิใช้งานเซิร์ฟเวอร์ฟรีตลอดชีพ (Always Free) ซึ่งคุ้มค่ามากๆ และเราสามารถตั้งค่าให้มีหน้าจอ (GUI) เหมือนคอมพิวเตอร์ Windows ทั่วไปให้คลิกใช้งานได้ง่ายๆ ด้วยครับ

---

## ขั้นตอนที่ 1: การสมัครสมาชิก
> **⚠️ คำเตือนสำคัญ:** ระบบกันสแปมของ Oracle ค่อนข้างเข้มงวดมาก แนะนำให้ใช้บัตรเครดิตจริง (บัตรเดบิตหรือ TrueMoney มักจะไม่ผ่าน) และกรอกข้อมูลให้ตรงกับบิลของบัตรเครดิตเป๊ะๆ ครับ

1. เข้าไปที่เว็บ [Oracle Cloud Free Tier](https://www.oracle.com/cloud/free/) แล้วคลิก **"Start for free"**
2. กรอกข้อมูลส่วนตัว อีเมล รหัสผ่าน ให้ครบถ้วน
3. **การเลือก Home Region:** สำคัญมาก! ให้เลือก **Singapore** หรือ **Japan (Tokyo/Osaka)** เพราะเป็นโซนเอเชียที่เชื่อมต่อได้รวดเร็ว (เลือกผิดจะเปลี่ยนทีหลังไม่ได้)
4. ยืนยันตัวตนด้วยบัตรเครดิต (ระบบจะหักเงินลองเช็คบัตรเล็กน้อย แล้วจะคืนให้ทีหลัง)
5. ถ้าระบบขึ้นว่าสำเร็จ ให้รอรับอีเมลยืนยันการสร้างบัญชี (อาจใช้เวลา 5-15 นาที)

## ขั้นตอนที่ 2: การสร้างเครื่อง VM (เซิร์ฟเวอร์)
เมื่อเข้าสู่ระบบ (Dashboard) ของ Oracle Cloud ได้แล้ว:
1. กดที่เมนูสามขีดซ้ายบน -> **Compute** -> **Instances**
2. กดปุ่ม **Create Instance**
3. ตั้งชื่อเครื่องอะไรก็ได้ในช่อง Name
4. **Image and Shape (สำคัญมาก!):**
   - กดปุ่ม `Edit` ตรง Image and Shape
   - **Image:** เปลี่ยนเป็น `Ubuntu` (แนะนำเวอร์ชัน 22.04)
   - **Shape:** เปลี่ยนเป็น `Ampere` (ARM) แล้วเลือก `VM.Standard.A1.Flex`
   - **ติ๊กเลือก OCPU:** เลื่อนแถบไปที่ `4` (สูงสุด) และ **RAM** เป็น `24GB` (สูงสุด) เพื่อให้ใช้งานได้ลื่นไหลที่สุด!
   *(หมายเหตุ: แถบสีเขียวจะเขียนว่า "Always Free Eligible" แปลว่าฟรีแน่นอน)*
5. **Networking:** ปล่อยค่าดั้งเดิมไว้ (แต่ให้จำไว้ว่ามี Public IP)
6. **Add SSH keys:** ให้เลือก **"Save private key"** และ **"Save public key"** ดาวน์โหลดเก็บไว้ในคอมพิวเตอร์ของคุณ **ห้ามทำหายเด็ดขาด**
7. **Boot volume:** ปรับขนาดตั้งไว้สัก `50 GB` - `100 GB` ก็เหลือเฟือแล้วครับ (อย่าใส่เต็ม 200 GB เพื่อเก็บโควต้าไว้สร้างเครื่องเล็กเพิ่มในอนาคต)
8. กดปุ่ม **Create** ด้านล่างสุด แล้วรอจนกว่าเครื่องจะขึ้นสถานะแถบสีเขียว (Running)

## ขั้นตอนที่ 3: เปิด Port สำหรับ Remote Desktop (3389)
บน Oracle ต้องเปิด Port สำหรับให้เราใช้โปรแกรม Remote Desktop เข้าไปควบคุมหน้าจอได้ครับ:

**ก. เปิดบนเว็บ Oracle:**
1. ในหน้าของ Instance ที่เพิ่งสร้าง ให้เลื่อนลงมากดคลิกที่ชื่อ **Subnet**
2. คลิกที่ **Security Lists** (มักจะชื่อ Default Security List...)
3. กด **Add Ingress Rules**
   - Source CIDR: `0.0.0.0/0`
   - Destination Port Range: `3389`
   - กด **Add Ingress Rules** เซฟให้เรียบร้อย

## ขั้นตอนที่ 4: การตั้งค่า Remote Desktop (VNC/RDP) ให้มีหน้าจอคลิกได้
เนื่องจาก Oracle เริ่มต้นมาจะมีแค่หน้าจอดำ เราต้องเข้าหน้าจอดำไปลงโปรแกรมหน้าจอ (GUI) ก่อนครั้งแรกครับ:

1. เปิดโปรแกรม `Windows PowerShell` ในคอมของคุณ
2. พิมพ์คำสั่งเพื่อเชื่อมต่อไปยังเครื่อง (ใช้ไฟล์ Private Key ที่โหลดมาจากตอนสร้างเครื่อง):
   ```bash
   ssh -i "C:\ที่เก็บไฟล์\ssh-key-2026-xx-xx.key" ubuntu@เลข_IP_ของเครื่อง_Oracle
   ```
   *(ตัวอย่าง: `ssh -i "C:\Users\son11\Downloads\ssh-key.key" ubuntu@123.45.67.89`)*
   ถ้าระบบถาม (yes/no) ให้พิมพ์ `yes` แล้ว Enter

3. เมื่อเข้ามาในจอดำ (หน้า Ubuntu) ให้ก๊อปปี้คำสั่งเหล่านี้ไปวางทีละบรรทัดเพื่อติดตั้งหน้าจอแบบเบา (XFCE4) และโปรแกรม RDP:
   ```bash
   sudo apt update && sudo apt upgrade -y
   sudo DEBIAN_FRONTEND=noninteractive apt install -y ubuntu-desktop
   sudo apt install -y xrdp
   ```
   *(รอโหลดติดตั้งนานนิดนึง ขึ้นอยู่กับสเปคเครื่อง)*

4. กำหนดรหัสผ่านสำหรับเข้าหน้าจอ Remote (สำคัญ! เอาไว้กรอกตอนเข้าจอ):
   ```bash
   sudo passwd ubuntu
   ```
   *(พิมพ์รหัสผ่านที่ต้องการ 2 รอบ ตัวหนังสือจะไม่แสดงขึ้นมาตอนพิมพ์ ถือว่าปกติครับ)*

5. เปิด Port 3389 ในเครื่อง Ubuntu (เฉพาะ Oracle ต้องทำ):
   ```bash
   sudo iptables -I INPUT 6 -m state --state NEW -p tcp --dport 3389 -j ACCEPT
   sudo netfilter-persistent save
   ```

6. สั่งเปิดโปรแกรม XRDP:
   ```bash
   sudo systemctl enable xrdp
   sudo systemctl restart xrdp
   ```

## ขั้นตอนที่ 5: วิธีเข้าใช้งานหน้าจอ!
ตอนนี้เครื่องเซิร์ฟเวอร์บน Oracle ของคุณมีหน้าจอพร้อมใช้งานแล้วครับ!
1. ในคอมพิวเตอร์ของคุณ (Windows) ให้ค้นหาโปรแกรมชื่อ **Remote Desktop Connection**
2. ตรงช่อง Computer ให้พิมพ์ **เลข_IP_ของเครื่อง_Oracle** ลงไป แล้วกด Connect
3. ถ้ามีหน้าต่างเตือนเด้งขึ้นมา ให้กด **Yes**
4. หน้าจอจะตัดเข้าสู่ระบบ Login ของ Ubuntu (หน้าต่าง XRDP สีฟ้าๆ)
   - **Username:** `ubuntu`
   - **Password:** `ใส่รหัสผ่านที่คุณเพิ่งตั้งไว้ในข้อ 4.4`
5. กด OK เข้าไปใช้งานหน้าจอ Desktop เหมือนคอม Windows ได้เลยครับ! จบปิ๊ง! 🚀
